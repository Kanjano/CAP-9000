# Assistant Project

## Directory Structure

- `src/` - Contains the main source code for the assistant.
  - `models/` - Contains the neural network models.
  - `data/` - Contains data processing scripts and datasets.
  - `utils/` - Contains utility functions and helpers.

- `tests/` - Contains unit tests for the assistant.

- `requirements.txt` - Lists the dependencies required for the project.

- `README.md` - Provides an overview of the project and instructions for setup.

- `main.py` - The entry point for the assistant application.
